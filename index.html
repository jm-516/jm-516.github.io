Bio

With a background in English, I specialize in creating content people can actually understand. My professional experience in a pharmaceutical library involves verifying controlled documentation, managing metadata, and ensuring strict SOP/GMP compliance.

This portfolio showcases my ability to apply that same attention to detail across different formats—including procedural SOPs, technical troubleshooting guides, and hardware quick start guides. My goal is to create documentation that is both accurate and useful.

Quick Start Guide for Bose Headphones

I developed a quick start guide for the Bose QuietComfort Ultra headphones after realizing the retail version lacked one. Designed for first-time users, this guide walks them through getting their device up and running quickly, covering essential setup, device pairing, basic controls, and common troubleshooting scenarios. I also used Figma to create a custom hardware diagram that clearly identifies the location and function of each control. This guide is an independent, unofficial document and is not affiliated with Bose Corporation.

SOP for Filing Documents in a GMP Library

Drawing on my experience as a pharmaceutical library assistant, I authored this model SOP for processing GMP library submissions. The document outlines a standardized workflow for submission verification, integrity checks, database entry, and filing practices. While inspired by real-world pharmaceutical standards, this is an independent project with all identifying details omitted or changed.

Tier 1 Troubleshooting Guide

I wrote this troubleshooting guide to help Windows 11 users resolve intermittent Wi-Fi disconnections after their device wakes from sleep mode. Designed for intermediate PC users, the guide provides step-by-step instructions for adjusting power settings, refreshing network connections, updating drivers, and escalating to manufacturer support.
